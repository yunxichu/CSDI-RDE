### CSDI模型原理

CSDI (Conditional Score-based Diffusion Models) 是一种基于条件分数的扩散模型，用于时序数据填补和预测：

1. **前向扩散过程**：逐步向数据添加噪声
2. **反向去噪过程**：学习从噪声中恢复数据
3. **条件机制**：利用观测到的历史数据进行条件生成

### GPR模块

`gpr_module.py` 实现了高斯过程回归：
- RBF核函数
- 超参数优化（最大化对数边际似然）
- 支持预测均值和不确定性（标准差）

### RDE-GPR方法原理

RDE (Random Dimension Ensemble) 结合 GPR 实现多步预测：

1. **随机嵌入**：从D维特征中随机选择L维组合
2. **集成预测**：采样s个不同组合
3. **GPR建模**：每个组合用高斯过程回归
4. **KDE融合**：用核密度估计合并多个预测结果

关键参数：
- `L`: 随机嵌入维度
- `s`: 采样组合数
- `trainlength`: 训练窗口长度
- `steps_ahead`: 预测步数

## 三、Baseline方法

位于 `baseline/` 目录下，包含多个对比基线：

### 1. NeuralCDE ([pm25_neuralcde_forecast.py](file:///home/rhl/Github/baseline/NeuralCDE-master/NeuralCDE-master/pm25_neuralcde_forecast.py))

神经控制微分方程方法：
- 将时序数据建模为连续时间动态系统
- 使用CDEint进行数值求解
- 适合不规则采样时间序列

### 2. GRU-ODE-Bayes ([pm25_gruodebayes_forecast.py](file:///home/rhl/Github/baseline/gru_ode_bayes-master/gru_ode_bayes-master/pm25_gruodebayes_forecast.py))

门控循环单元+常微分方程+贝叶斯推断：
- 结合神经ODE和GRU
- 引入贝叶斯Dropout估计不确定性
- 适合医疗等高缺失率场景

### 3. SSSD ([pm25_sssd_forecast.py](file:///home/rhl/Github/baseline/SSSD-main/SSSD-main/pm25_sssd_forecast.py))

结构化状态空间扩散模型：
- 结合S4结构（高效状态空间模型）
- 扩散过程用于生成式建模
- 支持条件生成

## 七、关键配置参数

### CSDI模型配置 (config/base.yaml)

```yaml
model:
  timeemb: 128        # 时间嵌入维度
  featureemb: 64      # 特征嵌入维度
  is_unconditional: false

diffusion:
  num_steps: 50       # 扩散步数
  schedule: "quad"    # 调度方式
  beta_start: 0.0001
  beta_end: 0.02
```

### RDE-GPR参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| L | 随机嵌入维度 | 4-6 |
| s | 采样组合数 | 50-200 |
| trainlength | 训练窗口 | 400-4000 |
| steps_ahead | 预测步长 | 1 |
| noise_strength | 噪声增强 | 1e-4 |

## 总结

你的项目实现了：
1. **CSDI** - 扩散模型用于时序数据补值
2. **RDE-GPR** - 随机维度集成高斯过程用于多步预测
3. **多个基线** - NeuralCDE、GRU-ODE-Bayes、SSSD用于对比

核心创新点在于将深度学习补值(CSDI)与传统统计学习预测(RDE-GPR)结合，形成两阶段预测框架。
