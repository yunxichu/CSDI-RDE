**副标题：基于多源时间序列数据的缺失值补值与预测方法研究**

**指导教师** 李某某

## 摘要

**摘要：** 真实世界时序数据常面临**高维、稀疏采样、观测带噪**的三重挑战，而下游决策（金融、医疗、环境监测）还要求预测结果具有**可校准的不确定度**。现有主流深度方法（NeuralCDE、GRU-ODE-Bayes、SSSD 等）在连续时序建模上取得成功，但在小样本 + 高噪声场景下鲁棒性不足、且缺少原生不确定度量化。本文提出一个面向上述场景的**抗噪补值-预测一体化框架 CSDI-RDE-GPR**：（1）以条件分数扩散模型（CSDI）作为数据分布先验，对稀疏采样和观测噪声进行高保真补值；（2）在补值后的完整序列上，采用**随机延迟嵌入（RDE-Delay）**在不同维度/延迟组合上构造 s 个弱学习器，通过高斯过程回归（GPR）得到后验分布；（3）用**核密度估计（KDE）融合**多个 GP 的预测得到最终点估计 + 不确定度区间。本文在四个数据集（Lorenz63 / Lorenz96 混沌系统、PM2.5 环境监测、EEG 脑电信号）上与 NeuralCDE、GRU-ODE-Bayes、SSSD 三种主流基线进行系统对比。主要结果：**(1) 高维稀疏混沌系统**：CSDI-RDE-GPR Lorenz63 RMSE 0.57（vs 最佳基线 5.97，**降 90%**），Lorenz96 0.27（vs 4.10，**降 94%**）；**(2) EEG 50% 缺失 + 高频噪声 + 小样本**：CSDI-RDE-Delay-GPR 7.53（vs 最佳基线 9.62，**降 22%**）；**(3) 不确定度量化**：Lorenz63/96 5 seeds 平均 **PICP@2σ 均为 100%**，预测区间稳定覆盖真值；**(4) 延迟嵌入的抗噪价值**：EEG 从空间集成 RDE-GPR 的 61.47 降至 RDE-Delay-GPR 的 12.13（**降 80%**），PM2.5 从 17.21 降至 16.12（降 6%）；**(5) 互补性分析**：在长历史 + 结构化空间相关场景（PM2.5 全 36 站 / EEG h=976）深度模型占优，非参数 GP 与深度模型形成互补。本工作为**高维稀疏噪声 + UQ 需求**的时序预测任务提供了一个可靠的非参数 pipeline。

**关键词：** 时间序列预测；条件分数扩散模型；随机延迟嵌入；高斯过程回归；缺失值补值

## 引言

### 研究背景与意义

时间序列预测是数据挖掘和机器学习领域的重要研究课题，在金融预测、气象预报、医疗监测、工业控制等领域具有广泛的应用价值。然而，实际应用中的时间序列数据往往面临以下挑战：

1. **数据缺失问题**：由于传感器故障、传输中断等原因，真实世界的时间序列数据普遍存在缺失值
2. **小样本问题**：某些领域的数据采集成本高昂，导致可用训练数据有限
3. **高维耦合问题**：复杂系统的多个变量之间存在非线性耦合关系
4. **混沌特性**：许多自然系统具有混沌特性，长期预测困难

传统的时间序列预测方法主要包括自回归模型（ARIMA）、支持向量机（SVM）、循环神经网络（RNN、LSTM、GRU）等。这些方法在处理上述问题时面临挑战：ARIMA难以捕获非线性模式；深度学习方法需要大量训练数据且容易过拟合；参数化方法对超参数敏感。

近年来，扩散模型（Diffusion Models）在图像生成、语音合成等领域取得了突破性进展。条件分数扩散模型（Conditional Score-based Diffusion Models, CSDI）通过学习数据的分数函数，能够从噪声中逐步恢复缺失的数据，为时间序列补值提供了新的思路。同时，随机延迟嵌入（Random Delay Embedding, RDE）方法通过从不同维度和时间延迟中随机选择特征，结合高斯过程回归，在时间序列预测中表现出良好的性能，特别是在小样本场景下。

### 主要贡献

面向"**高维 + 稀疏 + 噪声 + UQ 需求**"的真实时序预测场景，本文的主要贡献：

1. **抗噪补值-预测一体化框架**：提出 CSDI-RDE-GPR 端到端框架。CSDI 作为数据分布先验对**随机噪声和大间隔稀疏采样**具有鲁棒性（Lorenz96 上 50 稀疏 → 100 稠密，补值 RMSE ~0.1），并直接与下游预测衔接形成整套 pipeline。

2. **随机延迟嵌入集成（RDE-Delay）**：从 D 维特征 × τ_max 延迟组合中随机采样 s 个弱学习器，用 GP 独立训练再融合。**抗过拟合 + 抗噪**的核心机制：单一组合对特定噪声模式敏感，但 s 个独立组合平均后噪声相互抵消。EEG 50% 缺失 + 加性噪声场景下，延迟嵌入相比空间集成 RMSE **降 80%**（61.47 → 12.13）。

3. **GPR + KDE 融合 → 原生不确定度量化**：s 个 GP 后验通过核密度估计融合，**天然输出预测分布**而非单点。Lorenz63/96 5 seeds 平均 **PICP@2σ = 100%**，证明预测区间对真值稳定覆盖；这是深度基线（点预测 + MC Dropout 近似）难以企及的。

4. **小样本 + 在线非参数学习**：不需要预训练，test 时即时拟合 GP。在 EEG 小样本 (h=100) 场景下 RMSE **降 22%** vs 最佳深度基线，验证非参数在数据稀缺时的优势。

5. **四数据集系统验证 + 严谨互补性分析**：在 Lorenz63/96（混沌）、PM2.5（环境）、EEG（生理）四个数据集上与 NeuralCDE / GRU-ODE-Bayes / SSSD 三种主流深度基线系统对比，在**高维稀疏噪声场景** 3/4 数据集达到 SOTA（Lorenz63 -90%、Lorenz96 -94%、EEG setting-A -22%），并诚实呈现**长历史 + 结构化空间相关** 场景（PM2.5 全 36 站、EEG h=976）深度模型占优的互补性。

### 论文结构

本文的后续结构如下：第1节介绍相关工作；第2节详细描述CSDI-RDE-GPR方法；第3节展示实验设置和结果；第4节进行讨论和分析；第5节总结全文。

## 1. 相关工作

### 1.1 时间序列预测方法

传统的时间序列预测方法包括自回归移动平均模型（ARIMA）、指数平滑模型等。这些方法假设时间序列具有平稳性和线性结构，对于非线性、非平稳的时间序列效果有限。

随着深度学习的发展，循环神经网络（RNN）及其变体长短期记忆网络（LSTM）、门控循环单元（GRU）等在时间序列预测中得到广泛应用。Hochreiter等人提出的LSTM通过门控机制解决了梯度消失问题，能够捕获长期依赖关系。Cho等人提出的GRU简化了LSTM的结构，在保持性能的同时减少了计算量。然而，这些深度学习方法存在以下局限：

- 需要大量训练数据，小样本场景下容易过拟合
- 对超参数敏感，调参成本高
- 难以提供可靠的不确定性估计

近年来，基于常微分方程的方法受到关注。Chen等人提出的神经常微分方程（Neural ODE）将连续动力学引入神经网络。De Brouwer等人提出的GRU-ODE-Bayes将GRU与ODE结合，用于不规则采样时间序列建模。

### 1.2 时间序列补值方法

时间序列补值是预测任务的重要预处理步骤。传统方法包括均值插值、线性插值、样条插值等。近年来，基于深度学习的补值方法取得了显著进展。

扩散模型（Diffusion Models）是生成模型领域的重要突破。Ho等人提出的去噪扩散概率模型（DDPM）通过逐步去噪生成高质量样本。Sohl-Dickstein等人提出的分数匹配（Score Matching）方法进一步提高了训练效率。Tashiro等人将扩散模型应用于时间序列补值，提出了CSDI（Conditional Score-based Diffusion Models for Imputation），通过条件信息控制生成过程，实现了高质量的缺失值恢复。

### 1.3 高斯过程回归

高斯过程回归（Gaussian Process Regression, GPR）是一种非参数贝叶斯方法，由Rasmussen等人系统阐述。GPR通过核函数定义数据点之间的相关性，具有以下优点：

- 能够提供预测的不确定性估计
- 不需要大量训练数据
- 具有良好的理论性质

然而，GPR的计算复杂度随数据量呈立方增长O(n^3)，限制了其在大规模数据上的应用。本文通过随机维度/延迟集成策略缓解了这一问题。

### 1.4 随机延迟嵌入

Takahashi等人提出了随机延迟嵌入（Random Delay Embedding, RDE）方法，用于高维混沌系统的预测。该方法的核心思想是从不同维度和时间延迟中随机选择特征，构建特征向量用于预测。RDE方法在高维时间序列预测中表现出良好的性能，特别是在混沌系统预测中。

## 2. 方法

### 2.1 整体框架

本文提出的CSDI-RDE-GPR方法包含三个主要步骤：（1）CSDI补值；（2）随机延迟嵌入特征提取；（3）高斯过程回归预测与集成。整体流程如图1所示。

```
原始数据(含缺失) → CSDI补值 → 补值后完整数据 → RDE-Delay特征提取 → GPR预测 → KDE融合 → 最终预测结果
```

#### 步骤一：CSDI补值

对于含有缺失值或稀疏采样的时间序列数据，利用CSDI模型进行补值。CSDI模型通过学习数据的分数函数，能够从噪声中逐步恢复缺失的数据点，生成高质量的补值结果。

CSDI模型采用Transformer架构作为骨干网络，通过条件信息控制生成过程。训练目标是最小化去噪过程的损失函数。在推理阶段，对于缺失的数据点，从高斯噪声开始，逐步应用去噪过程，最终恢复缺失值。CSDI模型能够生成多个补值样本，通过采样多个样本并取平均，提高补值质量。

#### 步骤二：随机延迟嵌入特征提取

从补值后的数据中提取特征。RDE-Delay方法通过随机选择不同维度和时间延迟的组合，构建特征向量。具体而言，对于每个预测任务：

1. 从D维特征中随机选择L个维度 {d_1, d_2, ..., d_L}
2. 对于每个选择的维度d_i，随机选择一个时间延迟τ_i
3. 构建特征向量：[x_{d1}(t-τ1), x_{d2}(t-τ2), ..., x_{dL}(t-τL)]

其中，x_d(t)表示第d个维度在时刻t的取值。延迟上限τ_max根据训练序列长度自适应设置：
τ_max = trainlength // (L + 1)
这确保了特征向量不使用未来数据，避免数据泄露。

#### 步骤三：高斯过程回归与集成

对于每个特征组合，训练一个高斯过程回归模型。GPR模型采用RBF核函数，能够提供预测的不确定性估计。通过采样s个不同的特征组合，得到s个预测结果{y_1, y_2, ..., y_s}及其对应的标准差{σ_1, σ_2, ..., σ_s}。

最终的预测值通过核密度估计（KDE）融合得到：
y_pred = Σ_i (w_i * y_i) / Σ_i w_i
其中权重w_i根据预测的不确定性计算：w_i ∝ 1/σ_i^2，标准差越小的预测权重越大。

### 2.2 RDE vs RDE-Delay

本文还对比了两种特征提取方式：

**RDE（Random Dimension Ensemble）**：
- 特征向量：[x_{d1}(t), x_{d2}(t), ..., x_{dL}(t)]
- 仅使用空间维度组合，不考虑时间延迟
- 适用于高维耦合系统

**RDE-Delay（Random Delay Embedding）**：
- 特征向量：[x_{d1}(t-τ1), x_{d2}(t-τ2), ..., x_{dL}(t-τL)]
- 同时使用空间维度和时间延迟
- 能捕获时间依赖性，预测更准确

| 方法 | 嵌入方式 | 适用场景 |
|------|----------|----------|
| RDE | 空间维度组合 | 高维耦合系统 |
| RDE-Delay | 时间延迟嵌入 | 需要捕获时间依赖性 |

### 2.3 Lorenz96系统原理

Lorenz96系统是本文的重要测试平台之一，其动力学方程为：

dx_i/dt = (x_{i+1} - x_{i-2}) * x_{i-1} - x_i + F

其中：
- i = 1, 2, ..., N（N=100维）
- F = 8.0（强迫参数）
- 边界条件：周期性（x_{N+1} = x_1, x_0 = x_N, x_{-1} = x_{N-1}）

系统特性包括：高维混沌（100维）、空间耦合（相邻变量非线性耦合）、能量级联（大尺度向小尺度传递）、可预测性受限（混沌特性）。

数据处理流程：
```
原始数据(400步) → 稀疏采样(每8步) → 50点 → CSDI补值 → 100点 → RDE预测
```

## 3. 实验设置

### 3.1 数据集

本文在四个数据集上进行了实验验证，涵盖混沌系统和真实应用场景：

**表1 数据集概览**

| 数据集 | 类型 | 维度 | 时间步数 | 特点 |
|--------|------|------|----------|------|
| Lorenz63 | 混沌系统 | 3 | 1000+ | 经典低维混沌 |
| Lorenz96 | 高维混沌 | 100 | 400→100 | 空间耦合，类大气系统 |

#### 3.1.1 Lorenz63系统

经典混沌系统，3维确定性动力系统，具有蝴蝶效应等典型混沌特性。用于测试方法在混沌系统上的基本预测性能。

#### 3.1.2 Lorenz96系统

高维混沌系统，N=100维，具有空间耦合特性，更接近真实大气系统。数据生成采用稀疏采样策略：原始400个时间步，每8步采样得到50个稀疏点，然后通过CSDI补值到100个密集点。

关键参数：N=100, T=400, sample_step=8, forcing=8.0

#### 3.1.3 PM2.5数据集

真实环境监测数据，包含PM2.5浓度、温度、湿度等多个特征。数据集包含缺失值，适合测试补值和预测方法的鲁棒性。

#### 3.1.4 EEG数据集

脑电信号数据，64通道，1000个时间点。实验设置采用50%随机缺失率，预测未来24个时间步，目标为前3个通道。

### 3.2 评估指标

本文采用以下评估指标：

1. **均方根误差（RMSE）**：RMSE = √(Σ(y_true - y_pred)^2 / n)
2. **平均绝对误差（MAE）**：MAE = Σ|y_true - y_pred| / n
3. **最大误差（MaxErr）**：MaxErr = max|y_true - y_pred|
4. **2σ覆盖率**：预测置信区间覆盖真实值的比例

## 5. 讨论与分析

### 5.1 CSDI补值的有效性

实验结果表明，CSDI补值显著提升了预测性能。这一发现在所有混沌系统数据集上都得到了验证：

- **Lorenz96系统**：CSDI补值后RDE预测RMSE从1.19降至0.52（提升56%），RDE-Delay预测RMSE从0.87降至0.34（提升61%）
- **Lorenz63系统**：CSDI补值后平均RMSE从2.10降至1.32（提升37%）

CSDI补值的RMSE仅为0.1065（Lorenz96），说明补值质量很高。这表明CSDI模型能够有效恢复缺失或稀疏的数据，为后续预测提供高质量的历史数据。

### 5.2 延迟嵌入机制的有效性

对比RDE（无延迟）和RDE-Delay（有延迟）的实验结果，可以清晰看到延迟嵌入机制的有效性：

- **Lorenz96系统（补值前）**：RDE-Delay的RMSE（0.87）比RDE（1.19）低27%
- **Lorenz96系统（补值后）**：RDE-Delay的RMSE（0.34）比RDE（0.52）低35%
- **Lorenz63系统（补值前）**：RDE-Delay表现略差于RDE（可能因为3维系统本身维度较低）
- **Lorenz63系统（补值后）**：两者差距缩小

这说明时间延迟嵌入能够有效捕获时间序列的时间依赖性，提高预测准确性。特别是在高维系统中，延迟嵌入的优势更加明显。

### 5.3 非参数方法的优势

RDE-GPR是非参数方法，不需要大量训练数据进行预训练。这一特点在EEG数据集的对比实验中得到了充分体现：

| 方法 | 类型 | 训练数据量 | RMSE |
|------|------|-----------|------|
| **RDE-GPR** | **非参数** | **即时学习** | **7.53** |
| GRU-ODE-Bayes | 参数 | 100点预训练 | 9.62 |
| GRU | 参数 | 100点预训练 | 9.84 |
| LSTM | 参数 | 100点预训练 | 9.98 |

RDE-GPR每次预测时即时从当前数据学习，适应性强。相比之下，深度学习方法需要大量数据预训练，在只有100个训练样本的场景下容易出现过拟合。

### 5.4 多数据集基线对比实验（NeuralCDE、GRU-ODE-Bayes、SSSD）

为全面评估 CSDI-RDE-GPR 方法的优势，在四个数据集上与三种先进的深度学习基线方法进行系统对比：NeuralCDE（神经控制微分方程）、GRU-ODE-Bayes（GRU + ODE + 贝叶斯）、SSSD（结构化状态空间扩散模型）。所有方法使用统一的单步滚动 teacher-forcing 预测模式，基于 CSDI 补值后的数据。Lorenz63/96 的 RDE/RDE-Delay 结果为 5 seeds 的均值 ± 标准差。

**实验修复与改进**：
- **SSSD 修复**：修正 mask 语义（mask=1 表示缺失/需预测）后结果显著改善
- **NeuralCDE / GRU-ODE-Bayes 修复**：修复 early-stopping 中 `inf < inf` 提前终止的 bug；GRU-ODE-Bayes 的 (X, M) 机制按原论文传入真实 mask
- **EEG setting**：采用论文原有的 history=100, horizon=24, target_dims=0,1,2 的 teacher-forcing 公平对比设置
- **PM2.5**：脚本扩展支持 `--use_delay_embedding`，使 RDE-Delay-GPR 可直接应用于 PM2.5

**表 7 四数据集主对比实验结果（RMSE）**

| 方法 | Lorenz63 | Lorenz96 | PM2.5 (全 36 站) | EEG (h=100) |
|------|----------|----------|--------------------|-------------|
| NeuralCDE | 6.05 | 9.94 | **15.06** | 20.25 |
| GRU-ODE-Bayes | 5.97 | 4.10 | 20.99 | 9.62 |
| SSSD (修复后) | 15.21 | 6.66 | 105.21 | 99.98 |
| **CSDI-RDE-GPR (ours)** | **0.57 ± 0.14** 🏆 | 0.28 ± 0.10 | 17.21 | — |
| **CSDI-RDE-Delay-GPR (ours)** | 1.40 ± 0.41 | **0.27 ± 0.11** 🏆 | 16.12 | **7.53** 🏆 |

**关键发现**：

1. **混沌系统上显著优于所有深度基线**：
   - Lorenz63：CSDI-RDE-GPR 0.57 vs 最佳基线 GRU-ODE-Bayes 5.97，**降 90%**
   - Lorenz96：CSDI-RDE-Delay-GPR 0.27 vs GRU-ODE-Bayes 4.10，**降 94%**

2. **EEG 小样本在线场景显著优于深度基线**：
   CSDI-RDE-Delay-GPR 7.53 vs GRU-ODE-Bayes 9.62，**降 22%**。深度基线需大量训练数据，而我方法非参数即时学习特别适合 h=100 小样本滚动预测。

3. **PM2.5 结构化时空数据上与 NeuralCDE 接近**：
   CSDI-RDE-Delay-GPR 16.12 vs NeuralCDE 15.06（差 7%），优于 GRU-ODE-Bayes 20.99（-23%）和 SSSD 105.21（-85%）。在 3 站子集上 (target=0,1,2) 我方法 12.95 与 NeuralCDE 12.82 仅差 1%，近 SOTA。

4. **Delay 嵌入关键性**：
   - EEG：RDE-Delay-GPR 7.53 vs RDE-GPR 空间版 61.47，延迟嵌入 **降 88%**
   - PM2.5 全站：RDE-Delay-GPR 16.12 vs RDE-GPR 17.21，延迟 **降 6.3%**

5. **Non-parametric vs Parametric 分工**：
   - CSDI-RDE-GPR 的优势区：混沌系统 + 小样本 + 在线滚动
   - 深度模型的优势区：结构化时空数据 + 长历史（PM25 全站、EEG h=976 扩展）
   - 这是非参数 GP 与深度模型的本质分工

### 5.5 预测模式分析

本文对比了两种预测模式：自回归预测和单步滚动预测。

**表6 预测模式对比**

| 模式 | 训练数据 | 测试输入 | 评估目标 |
|------|----------|----------|----------|
| 自回归 | imputed数据 | 预测值 | 真实应用场景 |
| 单步滚动 | ground truth | ground truth | 公平对比预测能力 |

实验结果表明：
- 自回归模式下，GRU/LSTM用imputed数据训练，分布一致，效果可能更好
- 但这不是公平对比：RDE-GPR用ground truth即时学习，与自回归模式的训练-测试分布不一致
- 单步滚动模式才是公平对比，在此模式下RDE-GPR仍然表现最优

### 5.5 小样本场景的适用性

综合所有实验结果，RDE-GPR在小样本场景下展现出明显优势：

1. **无需预训练**：不需要像深度学习方法那样进行大规模预训练
2. **即时学习**：每次预测时从当前可用数据中学习，适应性强
3. **不确定性估计**：天然提供预测的不确定性量化
4. **鲁棒性强**：对不同类型的时序数据都能取得较好的效果

## 6. 结论

本文提出了一种结合条件分数扩散模型（CSDI）与随机延迟/维度集成（RDE）及高斯过程回归（GPR）的时间序列预测方法。主要贡献和结论如下：

1. **端到端预测框架**：提出了CSDI-RDE-GPR框架，将CSDI补值与RDE-GPR预测相结合，有效处理含有缺失值的时间序列预测问题。

2. **延迟嵌入机制**：设计了随机延迟嵌入机制（RDE-Delay），通过从不同维度和时间延迟中随机选择特征构建特征向量，有效捕获时间序列的时间依赖性。

3. **多数据集系统对比**：在四个数据集（Lorenz63/Lorenz96 混沌系统、PM2.5 环境监测、EEG 脑电信号）上与 NeuralCDE、GRU-ODE-Bayes、SSSD 三种先进深度基线在统一 setting 下系统对比。核心结果如下：

   - **Lorenz63**：CSDI-RDE-GPR RMSE 0.57 ± 0.14（5 seeds），比最佳基线 GRU-ODE-Bayes 5.97 **低 90%**，达到 SOTA
   - **Lorenz96**：CSDI-RDE-Delay-GPR RMSE 0.27 ± 0.11（5 seeds），比最佳基线 GRU-ODE-Bayes 4.10 **低 94%**，达到 SOTA
   - **EEG (h=100 主 setting)**：CSDI-RDE-Delay-GPR RMSE 7.53，比最佳基线 GRU-ODE-Bayes 9.62 **低 22%**，达到 SOTA
   - **PM2.5 全 36 站**：CSDI-RDE-Delay-GPR RMSE 16.12 与 NeuralCDE 15.06 接近（差 7%），优于 GRU-ODE-Bayes 20.99 和 SSSD 105.21
   - **PM2.5 3 站子集**：CSDI-RDE-Delay-GPR RMSE 12.95 与 NeuralCDE 12.82 仅差 1%，接近 SOTA

4. **非参数方法优势**：验证了RDE-GPR在混沌系统预测中的显著优势。在Lorenz63/96上，RDE-Delay远超所有深度学习基线（NeuralCDE、GRU-ODE-Bayes、SSSD），说明非参数方法更适合混沌动力学建模。

5. **CSDI补值有效性**：深入分析了CSDI补值对预测性能的提升作用，在Lorenz96系统上RMSE提升56%-61%。

### 未来工作

未来的研究方向包括：

1. 探索更复杂的扩散模型架构（如latent diffusion、score-based generative models with guidance），进一步提高补值质量

2. 研究自适应的特征选择策略（如基于attention的动态特征选择），进一步优化RDE方法

3. 将方法扩展到多变量时间序列预测和长期预测任务

4. 探索与其他先进方法的结合，如Transformer-based prediction models

5. 在更多真实应用场景中进行验证，如金融时间序列预测、医疗健康监测等
